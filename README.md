- 地图与侦察机路线融合（修复180°Bug）：小地图现已升级为800x400高分辨率（完美贴合游戏内20x10比例）·支持右键拖拽平移丶滚轮缩放，并显示底层网格代号·新增【侦察机航线规划】模块，点击开后后，在地图上左键双击（第一下定点，第二下定角度）即可极速生成航线·底层的方位角已修正为真实的360°算法·
- 诸元优先级颠倒与加粗：计划卡槽中，“仰角”现已移至“方位”前方，并且全程加粗高亮，符合火炮手装订诸元时的思维优先级。
- 火控UI重新排版布局：重构了左右炮实时解算的面板。现在仰角（巨大绿字）和方位（巨大黄字）极具视觉冲击力。而“目标已消灭”按钮被精简并放置在了面板右下角的弹种选择器上方，不喧宾夺主，且支持一键击灭联动目标并自动推挤补位下一个目标。

- 战术残缺碎片交会（破除半自动三角测距限制）：在战术应急面板中，不再强制要求每个基准点必须有方位和距离。你完全可以输入“A点56度”加上“B点10公里”，系统会调用全局函数库中的intersectRayAndCircle（射线与圆的交点计算）强行逆推坐标。
- 硬核网格向下取整（截断算法）：针对公里坐标，彻底抛弃了传统的Math.round（四舍五入），重写为Math.floor（向下取整）·完美模拟真实战术游戏引擎中的网格标定机制。
