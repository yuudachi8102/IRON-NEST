2026.6.22 V1.9.5
已解决问题：
1.战役第三关的战报中spotter后不加#号会导致无法正常识别的问题已经被解决。
2.有些时候游戏中会出现三角定位信息足够，但是判别式小于0的情况。比如射线不与圆相交， 两条射线反向平行等等。出现这种情况的时计算器会提示“[物理逻辑冲突/几何无解]”。
3.关于小地图。新版小地图已提升分辨率并支持放大缩小和拖动，以便于更直观的观察地图。
4.计划诸元卡中，方位角和仰角已经调换位置，以便于与火控台数据进行确认。
5.战术应急手动坐标输入面板中，半自动三角测距模式下每个基准点必须输入方位与距离的问题已解决。
6.战术应急手动坐标输入面板中，公里绝对坐标模式下自动四舍五入的问题已解决，现在该模式输入的目标将更加精确。

1.识别战报中的目标确认摧毁文本同步计算器状态。
2.点击击灭后，左上角小地图的目标状态应同步切换至灰色。
3.支持悬浮窗模式并覆盖在游戏记事本的位置。
